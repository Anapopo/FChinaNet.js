# fchinanet

fchinanet具有以下特性

+ [x] 使用Nodejs开发
+ [x] 全平台可用(win, mac, linux)
+ [x] 自动获取动态密码并登录
+ [x] 查看当前在线设备数
+ [ ] 下线当前设备

# 使用方法

+ 如果你是一名极客

```
1.使用`git clone`本项目
2.使用 `npm install` 或 `yarn` 来安装项目所需的依赖
3.解除文件尾函数loginChinaNet的注释
4.在`user.json`中填写用于登录掌上大学的`account`和`passwd`
5.执行 `node ./fchinanet.js`
6.等待程序自动登录...
```

+ 如果你是一名小白

1.稍后
